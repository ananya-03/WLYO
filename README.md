# WLYO - Who Let You Online?

WLYO is a gamified internet-era diagnosis wrapped in a procedural meme maze. It tests how well players remember meme images, audio clips, catchphrases, references, and timeline context, then turns their answers into a chaotic personality-style result.

Each run creates a new maze with seven checkpoints. Image checkpoints briefly show a meme, hide it, then ask the player to match the correct text answer from memory. Audio checkpoints play a short meme sound or reference and ask the player to identify what it is, where it belongs, or which era owns it.

Wrong answers do not end the game. Misses make the maze more insulting and push the final result older or more out of touch, but every player reaches the end and gets a verdict.

The final Vibe Report gives the player an estimated internet age, era label, roast, and four scores: Rizz, Aura, Sigma, and Era. The result is designed to feel like a shareable meme artifact, not a serious quiz.
